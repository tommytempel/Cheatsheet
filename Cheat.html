CSS Spacing

Property Functions Examples in Code
border It sets a space between two elements. border: 2px blue solid;
border-radius The border-radius CSS property rounds the corners of an element's outer border edge. You can set a single radius to make circular corners, or two radii to make elliptical corners. border-radius: 10px;
margin Need to look up the definition. margin: 50px;

CSS Psuedo-Classes

Psuedo-Classes Function Example in Code
:hover This sets the display of the mouse when hovering. a:hover{ text-decoration: underline; }
:visited Determines the display after a link has been visited. a:visited { color: #466995; }
:active The :active CSS pseudo-class represents an element (such as a button) that is being activated by the user. a:active { color: #DBE9EE; }
